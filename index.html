Сила Лоренца — уравнение (8.1)
F = Fe + FM = −eĒ − e[v̄, B̄]
Магнитная составляющая перпендикулярна скорости — не меняет модуль, только направление
Индукция соленоида — формула (8.9)
B = μ₀ · IL · N / l
μ₀ = 4π·10⁻⁷ Гн/м; N — число витков; l — длина соленоида; IL — ток соленоида
Критическая индукция и удельный заряд — формула (8.7)
e/m = 8U / (B²кр · r²a)
Вывод: при B = Bкр траектория электрона касается анода — eU = ½mv²φ, vφ = eBra/2m
Bкр = (2/ra) · √(2mU/e) → e/m = 8U / (B²кр · r²a)
Нахождение Bкр — метод касательной (рис. 8.1б)
Ступенька Id(IL) при B < Bкр — горизонт. участок; B > Bкр — спад
Iкр — ток соленоида в точке пересечения продолжения плато и касательной к крутому спаду. Bкр = k · Iкр, где k = μ₀·N/l.
Объёмный заряд — уточнённая формула (8.8)
B*кр = (2/ra) · √(2mU/e) / √(1−(r*/ra)²)
r* — положение потенциального минимума. При r* ≥ 0.2·ra поправка ≥ 5%. Кривая 2б рис. 8.1б — режим объёмного заряда.
Статистическая обработка — погрешность ⟨e/m⟩
σ⟨e/m⟩ = √[Σ((e/m)i − ⟨e/m⟩)² / (n·(n−1))]
Δ(e/m) = t(P=0.90, f=n−1) · σ⟨e/m⟩
Рассматриваем n значений (e/m)i при разных U как многократные наблюдения одной величины; f = n−1 степеней свободы.
